# **App Name**: Cyberfolio

## Core Features:

- Profile Display: Display Carolina Viana's name, and links to her GitHub and LinkedIn profiles.
- Skills Showcase: Showcase her skills including JavaScript, React, .Net, PHP, and Python.
- Artwork Display: Display artwork related to anime and the Cyberpunk 2077 theme.
- Generative Anime: Leverage a tool powered by generative AI to suggest relevant anime style elements, blending them contextually into her profile.

## Style Guidelines:

- Primary color: Neon Pink (#FF1493) to reflect the Cyberpunk theme.
- Background color: Dark Blue (#191970) for the base background color.
- Accent color: White (#FFFFFF) to stand out as highlights over the primary and background.
- Body and headline font: 'Space Grotesk' for a computerized, techy feel.
- Use glowing, tech-style icons to represent different skills and categories.
- Implement a grid-based layout with a futuristic, slightly asymmetrical feel.
- Add subtle animations, like glitch effects or neon glows, to enhance the Cyberpunk 2077 atmosphere.
